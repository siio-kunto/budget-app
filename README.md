# FinanceBird

**Personal finance & bookkeeping for sole proprietors in the DACH region.**

FinanceBird is a Progressive Web App (PWA) that gives freelancers and sole proprietors a simple, privacy-first tool for bookkeeping, invoicing, receipt management, and financial insights — all stored in the user's own workspace.

## Key Principles

- **Your data, your workspace** — FinanceBird connects to your existing tools. No proprietary database, no vendor lock-in.
- **Anti-Friction** — Every feature must remove more work than it adds. Max 3 taps, 10 seconds.
- **DACH-native** — Built for Swiss, German, and Austrian tax requirements from day one.

## Status

## Tech Stack

- Single-file PWA (HTML + Vanilla JS + CSS)
- No framework, no build system
- Cloudflare Workers for auth & API proxy
- GitHub Pages for hosting
